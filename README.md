# 🌄 Place Card UI Design

A clean and modern **Place Card UI** built using only **HTML and CSS**.
This project showcases a visually appealing card layout designed to highlight locations with images, tags, and descriptive content.

---

## 📌 Features

* 🖼️ Beautiful image-based card layout
* 🏷️ Category tags (e.g., Nature, Mountain)
* 📝 Title and description section
* 🎨 Clean and minimal UI design
* 📱 Responsive and centered layout

---

## 🚀 Technologies Used

* HTML5
* CSS3 (Flexbox & Styling)

---

## 🎯 Purpose of the Project

This project was created to practice and improve:

* Card-based UI design
* Image handling in layouts
* CSS styling and spacing
* Responsive design principles
